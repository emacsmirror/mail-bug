* mail-bug.el

Mail-bug is a fork of imapua.el (www.emacswiki.org/emacs/imapua.el)

** INSTALLATION
To enable it, put the directory in your load-path, and this in your
.emacs :

(require 'mail-bug)

** USAGE
Use M-x customize-group "mail-bug" to safely setup your prefs.
M-x mbug
